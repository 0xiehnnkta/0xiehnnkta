### [Nemesis Auditor](https://github.com/0xiehnnkta/nemesis-auditor)
[![Stars](https://img.shields.io/github/stars/0xiehnnkta/nemesis-auditor?style=flat-square)](https://github.com/0xiehnnkta/nemesis-auditor/stargazers) [![Forks](https://img.shields.io/github/forks/0xiehnnkta/nemesis-auditor?style=flat-square)](https://github.com/0xiehnnkta/nemesis-auditor/forks)
 
> *The Inescapable Auditor — iterative deep-logic security audit agent for [Claude Code](https://docs.anthropic.com/en/docs/claude-code)*
 
An agentic auditor that runs two complementary sub-agents in an alternating feedback loop — a **Feynman logic interrogator** and a **state inconsistency mapper** — until neither surfaces anything new. No pattern matching. Pure reasoning from first principles.
 
Most AI audit tools run a single pass. Nemesis converges: each pass feeds findings into the next, compounding coverage across up to 6 iterations until the loop goes silent.
 
```
Feynman Auditor          →   questions every assumption, every ordering choice,
                              every guard presence and absence across all functions
 
State Inconsistency      →   maps every coupled state pair (balance ↔ checkpoint,
Auditor                       stake ↔ rewardDebt), traces every mutation path,
                              finds every gap where one side updates without the other
 
Alternating loop         →   Feynman suspects become state targets.
                              State gaps become Feynman interrogation points.
                              Runs until convergence. Nothing survives both passes.
```
 
**Language-agnostic** — works on Solidity, Move, Rust, Go, C++, Python, TypeScript, or any codebase.
